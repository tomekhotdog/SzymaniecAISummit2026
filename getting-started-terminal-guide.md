# Getting Started with Claude Code

This guide will help you open a terminal and start Claude Code on your computer.

---

## For Mac Users

### Step 1: Open Terminal

**Option A - Using Spotlight (Recommended)**
1. Press `Command + Space` on your keyboard
2. Type `Terminal`
3. Press `Enter` or click on the Terminal app

**Option B - Using Finder**
1. Open Finder
2. Click on `Applications` in the sidebar
3. Open the `Utilities` folder
4. Double-click on `Terminal`

### Step 2: Create and Navigate to Your Project Folder

Once Terminal is open, type the following command and press `Enter`:

```
mkdir -p ~/Documents/NewProject1 && cd ~/Documents/NewProject1
```

This command creates the NewProject1 folder inside your Documents (if it doesn't already exist) and then navigates into it.

### Step 3: Start Claude

Type the following command and press `Enter`:

```
claude
```

Claude Code will now start and you can begin working!

---

## For Windows Users

### Step 1: Open Git Bash

**Option A - Using the Start Menu (Recommended)**
1. Click the Windows Start button (or press the `Windows` key)
2. Type `Git Bash`
3. Click on "Git Bash" when it appears in the search results

**Option B - Using the Desktop or Folder**
1. Right-click on your Desktop or inside any folder
2. Look for "Git Bash Here" in the menu
3. Click on it to open Git Bash in that location

### Step 2: Create and Navigate to Your Project Folder

## Troubleshooting

### "command not found: claude" error
This means Claude Code isn't installed yet. Please follow the installation instructions first, then come back to this guide.
